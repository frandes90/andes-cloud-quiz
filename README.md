# Andes Cloud — Salesforce Ecosystem Quiz

Quiz interactivo para profesionales del ecosistema Salesforce, desarrollado por **Andes Cloud**.

## ¿De qué se trata?

8 escenarios reales del mundo Salesforce donde el usuario debe identificar el producto correcto entre:

- Agentforce Sales
- Agentforce Service
- Tableau
- Slack
- Salesforce Industries
- Agentforce Field Service

## Tecnologías

- React 18
- Vite
- CSS en línea con Montserrat

## Cómo correrlo localmente

```bash
npm install
npm run dev
```
